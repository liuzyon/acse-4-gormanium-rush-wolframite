Detailed Description

This is a class that represent a CUnit.

The Cunit has some attributes and actions used in the circuit simulation.

◆ cal_con_tail()

void CUnit::cal_con_tail ( )

Calculate the flowrate of each component in concentrate and tailings streams.

Returns
void

use the current guess of the feed flowrates to calculate the flowrate of each component in both the concentrate and tailings streams. Each unit will recover 20% of the valuable material to the concentrate together with 5% of the waste.
